`config.yml`配置说明：

```yaml

# 站点信息
title: 站点标题
subtitle: 站点子标题
description: 站点描述
keywords: 站点关键字
# 作者信息
author: 作者名称
avatar: 头像图标
github: github地址
email: 邮箱
# 配置信息
summary_line: 首页文章行数
home_post_num: 首页文章数量
# 文件存储
theme: 模板文件夹
markdown: markdown文件夹
html: html文件夹
storage: 存储文件夹
# 文件监听
paths:
  - 监听文件
suffix:
  - 监听文件后缀
# 自定义信息
home_title: 主页子标题
archive_title: 归档页面子标题
tag_title: 标签页面子标题
category_title: 分类页面子标题
about_title: 关于我页面子标题

```
